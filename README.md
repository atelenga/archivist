# Archivist

Flask app for CSIRT

* Flask
* flask-script 
* WTForms 
* mongoengine 
* flask_mongoengine
* MongoDB

Order of deploying:

```commandline
virtualenv archivist/venv/
```

```commandline
pip install flask flask-script WTForms mongoengine flask_mongoengine
```

```commandline
source archivist/venv/bin/activate
```
